C++ ASF Library
by Constantin Titarenko, 2011

Solution contains C++ library and Qt application 
for writing and reading of either plain 
or gzipped ASF files.

Third party resources/libraries:
- icons from http://www.iconfinder.com/search/?q=iconset%3Aarzo
- gzipstream library from http://www.cs.unc.edu/Research/compgeom/gzstream/
- zlib library from http://zlib.net/
