R & C++ ASF Libraries
by Constantin Titarenko, 2011

Solution contains both R and C++ libraries for 
writing and reading of either plain or gzipped
ASF files. Also you can find ASF Viewer 
Qt application for viewing of ASF files. 

Please, explore <cpp> and <r> subfolders 
for more details.
